## [0.1.0-alpha.1] — 2026-06-21

First public alpha. Forge is a runtime control layer for coding agents —
deterministic task lifecycle, honest Git-based review, and local memory.

- Five-tool MCP surface: start, review, finish, submit outcome, expand output
- Baseline-tree capture and task-delta isolation from pre-existing dirty files
- Honest review: Git state, scope, readable content, and syntax only
- Local memory cards with finish-time creation and `/review-memory` maintenance
- OpenCode plugin: context governor, output compaction, system-prompt injection
- One-command install with SHA-256-verified bundles for Linux, macOS, Windows

Alpha-stage: breaking changes may occur before `1.0`.
